# Bashrc

To install bashrc

```sh
# Mac
cp bashrc/bashrc-mac-m2 ~/.bashrc
cat bashrc/il4-aliases >> ~/.bashrc
echo "source ~/.bashrc" > ~/.bash_profile


# linux-samson
cp bashrc/bashrc-linux-samson ~/.bashrc
echo "source ~/.bashrc" > ~/.bash_profile

# linux-UDRI
cat bashrc/bashrc-rocky-udri > ~/.bashrc

# linux-ray
cat bashrc/bashrc-linux-ray > ~/.bashrc

```
